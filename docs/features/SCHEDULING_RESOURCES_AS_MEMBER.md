# Scheduling Resources as a Member

This manual explains how members schedule resources on the calendar.

## 1. Before You Schedule

1. Confirm you are in the correct organization.
2. Confirm your required endorsements/currencies are current.
3. Check resource status and existing bookings.

## 2. Create a Reservation

1. Open `Calendar`.
2. Choose day/week/resource view.
3. Pick a time slot.
4. Select resource and event type.
5. Set start/end time and event details.
6. Save the event.

## 3. Understand Resource Statuses

1. `Available`:
Can be booked normally.
2. `Unavailable`:
Cannot be booked.
3. `Maintenance`:
Generally restricted to maintenance-capable roles.

## 4. Common Booking Restrictions

Booking can be blocked by:
1. Resource already booked for that time.
2. Missing required endorsement.
3. Enforced currency not current.
4. Overdue maintenance restrictions.
5. Organization schedule limit reached.

## 5. Member Tips

1. Use the calendar views to check conflicts before submitting.
2. Include enough buffer time after your flight - ensure that you consider time to postflight and refuel the aircraft before the next reservation begins.
3. Update or cancel bookings promptly when plans change.
