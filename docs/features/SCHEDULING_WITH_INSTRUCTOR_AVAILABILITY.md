# Scheduling with Instructor Availability

This manual explains how members request instructor time through availability slots.

## 1. What Instructor Availability Is

Instructors can publish availability slots on the calendar.

Members can request a time within those slots instead of booking directly against an instructor's schedule.

## 2. Member Workflow

1. Open `Calendar`.
2. Turn on `Show Availability` (if available in your organization).
3. Open an instructor availability event.
4. Click `Request Time Slot`.
5. Select requested start/end inside the availability window.
6. Add a clear purpose.
7. Submit the request.

## 3. Instructor Workflow

1. Receive an availability request notification.
2. Review requested time and purpose.
3. Accept or decline the request.
4. Provide reason when declining.

## 4. After Response

1. Accepted requests continue into normal scheduling flow.
2. Declined requests notify the member so they can request a different time.
3. Pending requests can be edited or canceled as needed.

## 5. Best Practices

1. Members: include clear training goals in request purpose.
2. Instructors: respond quickly to keep scheduling moving.
3. Use realistic block durations and avoid overbooking.
