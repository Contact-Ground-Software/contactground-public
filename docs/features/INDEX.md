# Contact Ground Feature Documentation

Welcome to the Contact Ground feature documentation. This index helps you navigate the product capability guides in the `docs/features` folder and find the technical overview for each major workflow area.

---

## 📚 What This Section Covers

These feature guides explain **what each capability does**, **where it fits in the platform**, and **how it supports club operations**.

If you are looking for step-by-step instructions by role, start with the [User Documentation Index](/docs/users/INDEX.md). If you want to understand how a product area works at a feature level, start here.
