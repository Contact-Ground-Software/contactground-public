# Maintenance Blocks

This manual explains maintenance blocks on the calendar.

## 1. What a Maintenance Block Is

A maintenance block reserves a resource for maintenance-related downtime.

It prevents normal member booking during that period.

## 2. Who Uses It

Maintenance-capable roles (typically owner/admin/operations) use maintenance blocks to protect maintenance windows.

## 3. Creating a Maintenance Block

1. Open `Calendar`.
2. Create an event for the target resource.
3. Choose event type `Maintenance` (or `Wash` when appropriate).
4. Set start/end times and save.

## 4. Overlap Behavior

If a maintenance block overlaps existing reservations, affected reservations can be cancelled and impacted members are notified.

## 5. Best Practices

1. Create blocks as promptly as possible.
2. Include clear notes in the event description.
3. Re-open resource availability promptly after maintenance completion.
