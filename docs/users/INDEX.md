# Contact Ground User Documentation

Welcome to the Contact Ground user documentation. This guide helps you find the right manual for your role and needs.

## 📚 Quick Start

**New to Contact Ground?**
- Start with [Getting Started](/docs/users/GETTING_STARTED.md)
- Start with [Pilot Scheduling Guide](/docs/users/members/PILOT_SCHEDULING_GUIDE.md) if you're a member/pilot
- Review [Administration](/docs/users/administration/ADMINISTRATION.md) if you're setting up an organization
- Check [Owner Configuration](/docs/users/owners/OWNER_CONFIGURATION.md) if you're an organization owner

## 💡 Tips for Using Documentation

1. **Use your browser's search** (Ctrl+F / Cmd+F) within documents to find specific topics
2. **Check cross-references** - many documents link to related guides
3. **Start with your role** - use the "by role" section above to find relevant docs
4. **Look at examples** - most guides include examples of good practices
5. **Check troubleshooting** - most guides have a troubleshooting section at the end
6. **Use the glossary** - standard terms are defined in [Glossary](/docs/users/GLOSSARY.md)

---

## 📞 Getting Help

If you can't find what you need in the documentation:
1. Check the relevant manual's troubleshooting section
2. Review the [Documentation Gap Analysis](/docs/DOCUMENTATION_GAPS.md) to see if it's a known gap
3. Contact your organization administrator
4. Reach out to Contact Ground support
