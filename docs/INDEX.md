## 🎯 Getting Started

### New Users
1. Review the [Pilot Scheduling Guide](/docs/users/members/PILOT_SCHEDULING_GUIDE.md)
2. Set up your [Notification Preferences](/docs/users/members/MEMBER_NOTIFICATION_PREFERENCES.md)
3. Familiarize yourself with [Checklists](/docs/users/operations/CHECKLISTS.md)

### New Administrators
1. Read [Administration](/docs/users/administration/ADMINISTRATION.md)
2. Review [Owner Configuration](/docs/users/owners/OWNER_CONFIGURATION.md)
3. Set up [Flight Operations](/docs/users/operations/FLIGHT_OPERATIONS.md) mode
4. Configure [Maintenance](/docs/users/operations/MAINTENANCE.md) workflows

### New Organizations
1. Start with [Owner Configuration](/docs/users/owners/OWNER_CONFIGURATION.md)
2. Add members using [Administration](/docs/users/administration/ADMINISTRATION.md)
3. Set up resources and checklists
4. Configure invoicing with [Accounting Playbook](/docs/users/accounting/ACCOUNTING_INVOICING_PLAYBOOK.md)

1. **Search the docs**: Use your browser's search (Ctrl+F / Cmd+F)

3. **Review troubleshooting**: Most guides have troubleshooting sections

5. **Contact support**: Reach out to your organization administrator or Contact Ground support
